git help - buscar ajuda
git init - inicia um repositorio com o git
git add - adicionar arquivo no git
git commit - gerar uma mudança
git config --global user.name "..." - configurar o nome
git config --global user.email "..." - configurar um e-mail
git status - verificar quais arquivos não estão commitados
git log - comando para verificar os commits realizados

- criar um repositorio no GitHub e seguir as instruções

git remote -v
git push -u origin master - criar a master no git
git push - fazer upload dos arquivos para o diretorio remoto
